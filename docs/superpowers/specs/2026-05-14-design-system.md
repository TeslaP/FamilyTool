# FamilyTool — Typography & Colour System v1

## Typography

### Inter (UI)
- Navigation, tables, forms, metrics, charts, controls
- Weights: 300 (hero), 400 (body), 500 (emphasis), 600 (buttons)

### Newsreader (Editorial)
- Login messages, AI reflections, monthly summaries, empty states, quotes
- Weights: 300, 400 (roman + italic)
- The contrast: "journal + operating system"

### Hierarchy
| Element | Font | Size | Weight | Spacing |
|---------|------|------|--------|---------|
| Hero metric | Inter | 72-88px | 300 | -0.04em |
| Section headings | Inter | 28-36px | 500 | — |
| Reflective text | Newsreader | 20-24px | 400 | — |
| Body | Inter | 16-18px | 400 | — |
| Table text | Inter | 14px | 400 | — |

## Colours
- App bg: stone-50 (#fafaf9)
- Cards: #ffffff with stone-200 borders
- Sidebar: stone-950 (#1c1917), hover: #292524
- Primary text: stone-950, Secondary: stone-600, Muted: stone-400
- Positive: #16a34a, Warning: #d97706, Negative: #dc2626
- Charts: monochrome stone (#57534e, #78716c, #a8a29e, #d6d3d1)

## Radius
- Cards: 20px, Inputs: 14px, Buttons: 14px, Small UI: 10px

## Shadows
- Card: 0 4px 20px rgba(28,25,23,0.06)
- Hover: 0 6px 24px rgba(28,25,23,0.08)

Horizon mark — layered lines in a circle, representing balance, perspective, and calm.
